📱 Labs & Foundations

Lab 1 · UI Basics
Jetpack Compose fundamentals & layout structures (Column, Row, Box).
📝 Reflection: This lab introduced me to Jetpack Compose basics and building user interfaces. I learned how to structure layouts using Column, Row, and Box, and how Material Design components create a professional look. This foundation was essential for all subsequent labs.
Lab 2 · Interaction & State
User interactions: clicks, text input, remember / mutableStateOf.
📝 Reflection: I learned how to handle user interactions like button clicks and text input. Understanding state management with remember and mutableStateOf was a key milestone. This lab taught me how to make apps responsive to user actions.
Lab 3 · Material Design
Theming, colour schemes, typography, shapes, and polished UI.
📝 Reflection: This lab focused on applying Material Design principles for a consistent and beautiful UI. I worked with color schemes, typography, shapes, and theming to create a visually appealing app that follows design guidelines.
Lab 4 · Navigation
Multi-screen apps: NavHost, routes, passing data between screens.
📝 Reflection: Navigation Compose was introduced in this lab, allowing me to create multi-screen apps. I learned about NavHost, routes, and passing data between screens. This was crucial for building StudyFlow's screen flow.
Lab 5 · Room Database
Local data persistence, SharedPreferences, offline storage.
📝 Reflection: I implemented local data persistence using Room Database and SharedPreferences. Learning how to store and retrieve data permanently was challenging but rewarding. This lab taught me the importance of offline-accessible data.
